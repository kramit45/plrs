# PLRS — Documentation Index

Generated and hand-written documentation for the Personalized Learning
Recommendation System (MCSP-232 capstone).

## Live (running app)

- **OpenAPI 3.0 spec** — `http://localhost:8080/v3/api-docs`
- **Swagger UI** — `http://localhost:8080/swagger-ui.html`
  (Auto-generated by SpringDoc; satisfies NFR-22.)

## Generated reference docs

| Tree                              | How to (re)build                              |
| --------------------------------- | --------------------------------------------- |
| `javadoc/`                        | `mvn javadoc:aggregate` (from repo root)      |
| `python/plrs-ml/`                 | `bash plrs-ml/scripts/build_docs.sh`          |
| `python/plrs-etl-worker/`         | `bash plrs-etl-worker/scripts/build_docs.sh`  |

The Java aggregate covers all four reactor modules (domain, application,
infrastructure, web). The Python trees use pdoc against the package
under `plrs_ml/` and `plrs_etl/` respectively.
